# 更多土地保护

请在plugins\Kingdoms\guis\en\structures\nexus\settings\relation-editor.yml中添加如下选项

```YAML
options:
  #珍珠传送...
  'AuspiceAddon:ender-pearl-teleport':
    enabled:
      condition: enabled
      name:
      lore:
    else:
      name:
      lore:    
  #信标增益
  'AuspiceAddon:beacon-effects':
    enabled:    
      condition: enabled
      name:
      lore:      
    else:     
      name:
      lore:    
  #鞘翅飞行
  'AuspiceAddon:elytra':
    enabled:
      condition: enabled
      name:
      lore:
    else:
      name:
      lore:    
```
